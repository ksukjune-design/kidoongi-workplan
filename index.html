사용 방법

📅 간트차트 — 가로축은 6/22(월)~9/6(일) 11주 일단위 달력, 세로축은 11개 부문. 각 BAR는 과업의 시작~종료일.

✏️ 편집 — BAR를 드래그하면 일정이 이동합니다. BAR 양끝을 잡아 늘리면 기간이 조절됩니다. BAR를 클릭하면 우측 편집 패널이 열립니다(과업명·일정·담당·성격·선행·메모 수정, 삭제).

🔗 연결 — 연결 모드 버튼을 켠 뒤 선행 과업 → 후행 과업 순으로 BAR를 클릭하면 화살표가 생성됩니다. 화살표를 클릭하면 삭제됩니다. (편집 패널에서도 선행 추가/삭제 가능)

✅ 진행 상태 — BAR를 클릭해 예정/진행중/완료를 설정하면, 바 하단 진행선·완료(✓)·상태점이 표시됩니다. 종료일이 지났는데 미완료면 지연(⚠·빨강 테두리)으로 자동 표시. 상단에 완료율·지연 수와 전체 진행바가 집계되고, 필터의 진행 칩으로 상태별로 볼 수 있습니다. 노란 "오늘" 선은 실제 오늘 날짜로 움직입니다.

🎯 임계경로 — 버튼을 켜면 9/1 런칭을 결정하는 과업 체인(가장 긴 선행 경로)이 분홍색으로 강조됩니다. 이 경로가 지연되면 런칭일이 밀립니다.

🗂 페이즈 — 상단 밴드는 6단계(킥오프·발굴 → 설계·확정 → 구축·개발 → 통합·연동 → QA·안정화 → 런칭·운영). 노란 점선 밴드는 이번 주(7/1).

🔎 상세도(3단계) — 전체(모든 과업) → 중간 요약(마일스톤·임계경로·핵심 장기 과업만) → 요약(부문별 1개 요약 BAR). 왼쪽 부문 헤더를 클릭하면 개별 부문도 접고 펼 수 있습니다.

⚙ 필터 — 조직/업무성격 칩을 눌러 특정 그룹만 강조. 검색창으로 과업명 필터.

💾 저장 — 뷰 설정(필터·접기·색상)은 각자 브라우저에 저장됩니다. 초기화로 기본값 복원.

🔗 실시간 공유(함께 이어서 작업) — 데이터는 Supabase에 실시간 공유됩니다. 링크만 있으면 누구나 열람·편집할 수 있고, 변경은 즉시 모두에게 반영됩니다(좌상단 🟢 실시간 연결). 별도 로그인/토큰이 필요 없습니다.

⬇ 엑셀 표 — 전체 과업(약 95개)을 CSV로 다운로드(엑셀에서 바로 열림).

✨ 1P 보고서(AI) — Gemini가 워크플랜을 16:9 슬라이드 1장으로 압축 생성(전체 나열 X, 부문·마일스톤·임계경로 요약). 📥 PPTX 다운로드(실제 .pptx 슬라이드) 또는 🖨 PDF(16:9)로 출력. 🔄 재생성 가능. 키는 팀 공유(자동) 또는 🔑로 개인 키 입력.
